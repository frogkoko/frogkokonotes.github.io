What are FANBOYS?

El término "FANBOYS" se usa para referirse a la lista de conjunciones coordinantes que son usadas para hacer una oración compuesta de dos oraciones independientes. Usando la palabra FANBOYS es un modo fácil para recordar las conjunciones coordinantes.

F A N B O Y S stands for: For, And, Nor, But, Or, Yet, So

These coordinating conjunctions connect words, phrases, or clauses of equal importance in a sentence.
